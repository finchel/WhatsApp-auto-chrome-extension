Privacy Policy

WhatsApp Holiday Greeter Chrome Extension

Last Updated: March 29, 2026

Summary: This extension stores all data locally on your device. It does not collect, transmit, or share any personal information. No analytics, no tracking, no external servers.

1. What Data the Extension Accesses

This extension operates exclusively on the web.whatsapp.com website. When you click on a 1:1 chat in WhatsApp Web, the extension reads the contact's display name from the visible page content to translate it into Hebrew and generate a personalized greeting.

The extension does not read your messages, conversation history, media, or any other WhatsApp content.

2. What Data is Stored

The extension stores the following data locally on your device using Chrome's built-in storage API (chrome.storage.local):

  • Custom greeting templates (male and female) that you configure
  • A mapping of English names to Hebrew names and genders that you manually enter through the unknown-name prompt

This data never leaves your device.

3. What Data is Collected or Transmitted

None. This extension:

  • Does NOT collect any personal information
  • Does NOT transmit any data to external servers
  • Does NOT use analytics, tracking pixels, or telemetry
  • Does NOT use cookies
  • Does NOT make any network requests (all name translation uses a bundled local dictionary)
  • Does NOT access, read, or store your WhatsApp messages or conversations
  • Does NOT access your WhatsApp contacts list beyond the single name visible when you click a chat

4. Third-Party Services

This extension uses no third-party services, APIs, or SDKs. It contains no AI components. All processing happens locally in your browser.

5. Permissions Explained

  • "storage" — Used to save your greeting templates and custom name mappings locally on your device.
  • Host permission for web.whatsapp.com — Required to run the content script that detects chat clicks and inserts greeting messages on the WhatsApp Web page.

6. Data Retention and Deletion

All data is stored locally in your browser. You can delete all extension data at any time by:

  • Removing individual saved names from the extension popup
  • Uninstalling the extension (all data is automatically deleted)
  • Clearing Chrome extension storage via chrome://settings

7. Import/Export Feature

The extension allows you to export your saved name mappings as a JSON file and import name mappings from a JSON file. These files are stored on your local device and are never uploaded anywhere. You control who you share exported files with.

8. Children's Privacy

This extension is not directed at children under 13 and does not knowingly collect data from children.
